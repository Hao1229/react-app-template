## How To Start

### Init

`pnpm install`

### Development

`pnpm serve`

Open [http://localhost:3000/](http://localhost:3000/) in the browser.

### Build

`pnpm build`

Build `dist` folder then you can deploy it.

### Commit

`pnpm commit`

Use `cz-conventional-changelog` to rule your commit message.
